2. 버켓에 로컬의 모델 업로드 <br>

* DRfC환경에서 aws configure 명령어를 통해 credential 검증 <br>
  (로컬학습환경 구성 과정에서 AWS 가입 후 발급한 액세스키, 비밀 액세스키 입력)
  ```sh
  aws configure
  ```
* 로컬 환경에서 "aws s3 ls" 명령어를 입력했을 때 AWS S3 서비스에서 생성한 버킷의 리스트가 출력된다
  ```sh
  aws s3 ls
  ```
